Per te instaluar aplikacionin duhet te jete i instaluar python ne pc.
Krijo nje virtual envoirment me komanten:
Windows: python -m venv .venv
Mac/Linux: python3 -m venv .venv
Instalo paketat e nevojshme me komanden: pip install -r requirements.txt
Pasi te jene instaluar te gjitha paketat me starto programin me komanden: python manage.py runserver

Url per faqen kryesore: http://localhost:8000/quiz
